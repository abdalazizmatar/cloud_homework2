What is CI/CD?

CI/CD means Continuous Integration and Continuous Deployment. It is a process used in cloud computing to automatically test and deploy code. When developers push changes to the repository, automated tools run checks and deploy the application. This helps developers release updates faster and more reliably.

My Pipeline

When I push my code to GitHub, GitHub Actions automatically runs a workflow. It checks the project files and then deploys the website to GitHub Pages. After the process finishes successfully, the website becomes live online.
